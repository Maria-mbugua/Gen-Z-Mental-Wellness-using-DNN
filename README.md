# Studio Website

This project is a studio website built with modern web technologies.

## Link

[Live Website](https://studio-website-psi.vercel.app/)

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

To get started with this project locally, follow these steps:

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:
   ```sh
   cd <YOUR_PROJECT_NAME>
   ```

3. Install dependencies:
   ```sh
   npm install
   ```

4. Start the development server:
   ```sh
   npm run dev
   ```

## Editing

You can edit the code using your preferred IDE. The project is set up with hot module replacement for a smooth development experience.
